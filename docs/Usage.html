4 Usage

You must do the required setup before this package will work properly. This setup includes installing the required dependencies and making the required configurations.

See Dependencies

See Configuration

4.1 Ethernet Connection

M-x internet-ethernet-connect will make an Ethernet connection.

M-x internet-ethernet-disconnect will disconnect the Ethernet connection.

4.2 Ethernet and OpenVPN Connection

M-x internet-ethernet-openvpn-connect will make an Ethernet and OpenVPN connection. You will be prompted for the OpenVPN configuration file to use when connecting.

M-x internet-ethernet-openvpn-disconnect This will disconnect the Ethernet and OpenVPN connection.

4.3 Wireless Network Configuration

M-x internet-add-wireless-configuration will add a wpa_supplicant configuration file for a wireless network. You will be prompted for a network name, network SSID and network password.

M-x internet-remove-wireless-configuration will remove a wpa_supplicant configuration file. You will be prompted for which configuration file to remove.

4.4 Wireless Connection

M-x internet-wireless-connect will make a wireless connection. You will be prompted for the wpa_supplicant configuration file to use when connecting.

M-x internet-wireless-disconnect will disconnect the wireless connection.

4.5 Wireless and OpenVPN Connection

M-x internet-wireless-openvpn-connect will make a wireless and OpenVPN connection. You will be prompted for the wpa_supplicant configuration file and the openvpn configuration file to use when connecting.

M-x internet-wireless-openvpn-disconnect will disconnect the wireless and OpenVPN connection.
