2 Dependencies

2.1 Ethernet Connection

If you are connecting using Ethernet you need to have these programs installed:

  • ip-link
  • dhclient
  • sudo

2.2 Ethernet OpneVPN Connection

If you are connecting using Ethernet and OpenVPN you need to have these programs installed:

  • ip-link
  • dhclient
  • openvpn
  • sudo

2.3 Wireless Connection

If you are connecting using wireless you need to have these programs installed:

  • rfkill
  • ip-link
  • wpa_supplicant
  • dhclient
  • sudo

2.4 Wireless Openvpn Connection

If you are connecting using wireless and OpenVPN you need to have these programs installed:

  • rfkill
  • ip-link
  • wpa_supplicant
  • dhclient
  • openvpn
  • sudo

2.5 Creating Wireless Configurations

If you are going to use this package to create wireless configurations then you need to have these programs installed:

  • wpa_passphrase
  • sudo
