3 Configuration

3.1 Configure for Ethernet connecting

Set the internet-ethernet-interface variable to the ethernet interface you will be using. You can look up your wireless interface by running the command ip link show. It will start with enp or eth.

Here is an example of setting the variable: 

(setq internet-ethernet-interface "enp3s0")

3.2 Configure for Ethernet and OpenVPN connecting

Set the internet-ethernet-interface variable to the ethernet interface you will be using. You can look up your wireless interface by running the command ip link show. It will start with enp or eth.

Here is an example of setting the variable: 

(setq internet-ethernet-interface "enp3s0")

Set the internet-openvpn-config-directory variable to the directory where you have your openvpn configuration files.

Here is an example of setting the variable: 

(setq internet-openvpn-config-directory
      "/home/user-name/openvpn/configurations/")

3.3 Configure for wireless connecting

Set the internet-rfkill-device-id variable to the wireless device id for the device you will be using. You can look up your wireless device id by running the command rfkill.

Here is an example of setting the variable: 

(setq internet-rfkill-device-id "0")

Set the internet-wireless-interface variable to the wireless interface you will be using. You can look up your wireless interface by running the command ip link show. It will start with wlp or wlan.

Here is an example of setting the variable: 

(setq internet-wireless-interface "wlp4s0")

Set the internet-wpa-supplicant-config-directory variable to the directory where you have your wpa_supplicant configuration files.

Here is an example of setting the variable: 

(setq internet-wpa-supplicant-config-directory
      "/etc/wpa_supplicant/")

Set the internet-wpa-supplicant-driver variable to your wpa_supplicant driver. For information on the driver options refer to the wpa_supplicant manual’s available drivers section with command man wpa_supplicant.

Here is an example of setting the variable: 

(setq internet-wpa-supplicant-driver "nl80211")
